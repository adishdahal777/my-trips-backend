# AGENTS.md

## Mission
Laravel API backend for a trip-sharing app: users create trips with route stops, expenses, photos, and notes; trips can be public with likes/comments from other users.

## Toolchain
| Task | Command |
|---|---|
| Install | `composer install && npm install` |
| Dev | `composer dev` |
| Test | `composer test` |
| Lint | `./vendor/bin/pint` |
| Migrate | `php artisan migrate` |

## Judgment Boundaries
- NEVER commit `.env` or the sqlite DB file.
- ASK FIRST before adding dependencies or changing auth flow (Sanctum/Socialite/OTP).
- ALWAYS test after model or migration changes.

## Non-Standard Tooling
- API docs generated by `dedoc/scramble` from route/FormRequest introspection — no manual OpenAPI file to maintain.
- Three auth paths coexist: Sanctum token, Google OAuth (Socialite), OTP email (`app/Mail/OTPMail.php`).
